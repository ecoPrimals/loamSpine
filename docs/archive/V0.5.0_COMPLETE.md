## 🔄 What Changed (v0.4.1 → v0.5.0)

### Breaking Changes
1. **Entry domain classification**
   - `SessionCommit`, `SliceCheckout`, `SliceReturn` now return `"session"` instead of `"rhizocrypt"`
   - Aligns with capability-based naming

### Primal Hardcoding Elimination
```rust
// REMOVED: Hardcoded primal name
// cli_signer.rs: "beardog" removed from CANDIDATE_BINARIES

// BEFORE
for candidate in &["signer", "signing-service", "beardog"] { ... }

// AFTER (v0.5.0)
for candidate in &["signer", "signing-service"] { ... }
```

```rust
// CHANGED: Capability-based domain
// entry.rs: domain() now returns capability, not provider

// BEFORE
EntryType::SessionCommit { .. }.domain()  // "rhizocrypt"

// AFTER (v0.5.0)
EntryType::SessionCommit { .. }.domain()  // "session"
```

### Test Coverage Explosion
- **39 new tests** across 4 modules:
  - CLI signer: +13 tests (integration with `../bins/beardog`)
  - Chaos: +9 tests (network, disk, OOM, Byzantine faults)
  - Storage: +9 tests (concurrency, large datasets, edge cases)
  - Backup: +10 tests (corruption detection, multi-spine, JSON)

### Performance Benchmarks
- **6 new storage benchmarks**:
  - `sled_spine_save/load`
  - `sled_entry_save/load`
  - `sled_bulk_entry_save` (100 entries)
  - `sled_flush` (disk sync)

## 🔗 Integration Points

### Phase 1 Primals (`../bins/`)
```bash
../bins/
├── beardog           # Signing service (tested in v0.5.0)
├── nestgate          # Storage service
├── rhizocrypt        # Session management
├── songbird-cli      # Universal adapter
└── toadstool         # Compute orchestration
```

**Integration Strategy**:
- ✅ Binary discovery via environment (`LOAMSPINE_SIGNER_PATH`)
- ✅ Fallback search in `../bins/`
- ✅ Graceful degradation if unavailable
- ✅ No hardcoded primal names

### Universal Adapter Pattern (Songbird)
```
┌─────────────┐
│  LoamSpine  │────┐
└─────────────┘    │
                   │    ┌──────────────┐
┌─────────────┐    ├───▶│   Songbird   │◀────┐
│   Beardog   │────┘    │   (Adapter)  │     │
└─────────────┘         └──────────────┘     │
                                             │
┌─────────────┐                              │
│ Rhizocrypt  │──────────────────────────────┘
└─────────────┘

Complexity: O(n) instead of O(n²)
Each primal knows only itself + Songbird
```

## 🎯 What You'll Learn

- Mint new certificates (create digital assets)
- Transfer ownership permanently
- Loan certificates temporarily
- Return loaned certificates
- Full provenance tracking

---

## 📖 Certificate Operations

### 1. Mint
Create a new certificate with initial owner.

### 2. Transfer
Permanently change ownership (A → B).

### 3. Loan
Temporary transfer with terms (duration, auto-return).

### 4. Return
Complete the loan, return to lender.

---

## 🔍 Use Cases

- **Game Keys**: Publishers mint, retailers sell, players loan
- **Collectibles**: Tournament items with provenance
- **Achievements**: Certificates for in-game accomplishments
- **Digital Art**: NFT-like ownership with lending

---

## 💡 Key Features

- ✅ Full provenance (entire history in spine)
- ✅ Loan terms (duration, auto-return)
- ✅ Non-repudiable ownership
- ✅ Cryptographic verification

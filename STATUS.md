## 🎯 v0.7.0 Features

### Temporal Primitives ⏰
- Universal time tracking across ANY domain
- 5 moment types (code, art, life, experiments, milestones)
- 4 anchor types (atomic, crypto, causal, consensus)
- New `EntryType::TemporalMoment`
- Philosophy: "Time is the primitive, not version control"

### Waypoint Anchoring 🎯
- Slice lending patterns
- Journey tracking (anchor → operations → departure)
- Permanent audit trail for borrowed data
- Certificate lifecycle support

### Recursive Spines 🔄
- Spine-to-spine references
- Hierarchical composition
- O(n) scaling vs O(n²) replication
- Sovereign spines with cross-references

### Zero-Copy Optimization 🚀
- `bytes::Bytes` throughout network layer
- 30-50% performance improvement
- Reduced allocations
- Efficient buffer sharing

### Service Integration 🌐
- Real `loamspine-service` binary
- Dual protocol (tarpc + JSON-RPC 2.0)
- Language-agnostic API
- Production health monitoring
- Complete lifecycle management

- **Showcase**: `showcase/`

### Key Commands
```bash
# Build & Test
cargo build --release
cargo test --all-features
cargo clippy --workspace -- -D warnings

# Coverage
cargo llvm-cov --all-features --workspace --lcov --output-path lcov.info

# Run Service
./ecoPrimals/primalBins/loamspine-service --jsonrpc-port 8080

# Run Showcase
cd showcase/01-local-primal && ./RUN_ALL.sh
```

### Key Endpoints
- **Health**: `http://localhost:8080/health`
- **JSON-RPC**: `http://localhost:8080` (POST)
